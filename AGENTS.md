# Style

A simple web page/app themeable style system.

Example usage is demonstrated in

* [index.html](index.html)
* [star-wars.css](star-wars.css)

To switch themes, point `style.css` at the desired theme file.
For example

```bash
ln -s star-wars.css style.css
```

All supported HTML elements are defined in [index.html](index.html).
